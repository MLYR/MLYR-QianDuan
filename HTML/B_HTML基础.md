## HTML 基础- 4个实例

​		当涉及到HTML的基础实例，通常会涉及一些最常见和基本的HTML标签，比如标题、段落、链接和图像等。以下是四个HTML基础实例：

1. **标题（Headings）：**
   使用`<h1>`到`<h6>`标签来定义标题，其中`<h1>`表示最高级别的标题，`<h6>`表示最低级别的标题。标题用于给文档结构化，并且标题的级别决定了其重要性。

```html
<!DOCTYPE html>
<html>
<head>
    <title>HTML基础示例 - 标题</title>
</head>
<body>
    <h1>这是一级标题</h1>
    <h2>这是二级标题</h2>
    <h3>这是三级标题</h3>
</body>
</html>
```

2. **段落（Paragraphs）：**
   使用`<p>`标签来定义段落。段落用于组织和呈现文本内容。

```html
<!DOCTYPE html>
<html>
<head>
    <title>HTML基础示例 - 段落</title>
</head>
<body>
    <p>这是第一个段落。</p>
    <p>这是第二个段落。</p>
</body>
</html>
```

3. **链接（Links）：**
   使用`<a>`标签来创建链接，通过`href`属性指定链接的目标URL。

```html
<!DOCTYPE html>
<html>
<head>
    <title>HTML基础示例 - 链接</title>
</head>
<body>
    <p>这是一个链接到百度的网站：<a href="https://www.baidu.com">百度</a></p>
</body>
</html>
```

4. **图像（Images）：**
   使用`<img>`标签来插入图像，通过`src`属性指定图像的URL，`alt`属性用于在图像无法显示时显示替代文本。

```html
<!DOCTYPE html>
<html>
<head>
    <title>HTML基础示例 - 图像</title>
</head>
<body>
    <img src="example.jpg" alt="示例图像">
</body>
</html>
```

这些是HTML的四个基础实例，它们是构建网页的最基本组成部分。通过组合和扩展这些基础实例，可以创建出更加复杂和丰富的网页。随着CSS和JavaScript等技术的结合使用，可以为网页增加更多的样式和交互效果。

## HTML 元素

​		HTML元素是构成HTML文档的基本组成单元，它们由一对标签组成，通常包括一个开始标签和一个结束标签。开始标签和结束标签之间的内容被称为元素的内容，可以是文本、图像、链接等。HTML元素用于描述网页的结构和内容，使得网页可以被浏览器正确解析和显示。

### HTML 元素语法

- HTML 元素以**开始标签**起始
- HTML 元素以**结束标签**终止
- **元素的内容**是开始标签与结束标签之间的内容
- 某些 HTML 元素具有**空内容（empty content）**
- 空元素**在开始标签中进行关闭**（以开始标签的结束而结束）
- 大多数 HTML 元素可拥有**属性**

## HTML 属性

​		HTML属性是用于提供额外信息或配置HTML元素的特性。属性可以添加到HTML标签中，通过属性，我们可以控制元素的行为和样式，以及与其他元素之间的交互。属性通常以名称-值对的形式出现，名称和值之间使用等号 "=" 连接。

- HTML 元素可以设置**属性**
- 属性可以在元素中添加**附加信息**
- 属性一般描述于**开始标签**
- 属性总是以名称/值对的形式出现，**比如：name="value"**。

> html属性参考网址 https://www.runoob.com/tags/html-reference.html

以下是一些常见的HTML属性示例：

1. **id属性：**
   `id`属性用于给HTML元素指定一个唯一的标识符，方便在JavaScript或CSS中对该元素进行操作或样式设置。

```html
<div id="header">这是页眉</div>
```

2. **class属性：**
   `class`属性用于为HTML元素定义一个或多个样式类，可以通过CSS选择器来选中这些元素并应用样式。

```html
<p class="highlight">这是一个带有highlight类的段落。</p>
```

3. **src属性：**
   `src`属性用于指定图像、音视频或iframe的来源URL。

```html
<img src="example.jpg" alt="示例图像">
```

4. **href属性：**
   `href`属性用于指定链接的目标URL。

```html
<a href="https://www.example.com">这是一个链接</a>
```

5. **alt属性：**
   `alt`属性用于在图像无法显示时显示替代文本，也可用于辅助技术（如屏幕阅读器）读取。

```html
<img src="example.jpg" alt="示例图像">
```

6. **width和height属性：**
   `width`和`height`属性用于指定图像、表格或其他元素的宽度和高度。

```html
<img src="example.jpg" alt="示例图像" width="300" height="200">
```

7. **disabled属性：**
   `disabled`属性用于禁用表单元素，阻止用户输入或选择。

```html
<input type="text" disabled>
```

8. **target属性：**
   `target`属性用于指定链接在何处打开，如"_blank"表示在新窗口打开链接。

```html
<a href="https://www.example.com" target="_blank">在新窗口打开链接</a>
```

以上只是HTML属性的一些常见示例，HTML提供了许多属性用于控制元素的行为和样式。通过合理地使用属性，可以实现丰富多样的网页效果和交互。

## HTML标题

在HTML中，标题是通过使用标题标签（Heading Tags）来表示的。标题标签提供了不同级别的标题，从最高级别的标题`<h1>`到最低级别的标题`<h6>`。这些标签帮助构建页面的结构，并使内容具有层次感和重要性。以下是HTML标题标签的示例：

```html
<!DOCTYPE html>
<html>
<head>
    <title>HTML标题示例</title>
</head>
<body>
    <h1>这是一级标题</h1>
    <h2>这是二级标题</h2>
    <h3>这是三级标题</h3>
    <h4>这是四级标题</h4>
    <h5>这是五级标题</h5>
    <h6>这是六级标题</h6>
</body>
</html>
```

在上面的示例中，使用了`<h1>`到`<h6>`标签来创建不同级别的标题。这些标签的重要性逐渐减少，即`<h1>`是最高级别的标题，`<h6>`是最低级别的标题。

在使用标题标签时，应该遵循逻辑顺序，不要跳跃级别。例如，不要在没有`<h2>`标题的情况下直接使用`<h3>`标题。良好的标题结构有助于优化页面的可读性和搜索引擎优化（SEO）。

## HTML 水平线

在HTML中，水平线（Horizontal Line）可以通过使用`<hr>`标签来创建。`<hr>`标签用于在文档中插入一条水平分隔线，通常用于分隔不同部分的内容或创建视觉分隔。

下面是一个简单的示例，演示如何使用`<hr>`标签创建水平线：

```html
<!DOCTYPE html>
<html>
<head>
    <title>HTML水平线示例</title>
</head>
<body>
    <h2>第一部分</h2>
    <p>这是第一部分的内容。</p>

    <hr> <!-- 创建水平线 -->

    <h2>第二部分</h2>
    <p>这是第二部分的内容。</p>
</body>
</html>
```

在上面的示例中，`<hr>`标签在第一部分和第二部分之间创建了一条水平分隔线。

`<hr>`标签是自闭合标签，不需要闭合标签。默认情况下，它会在页面上创建一条水平线，但您可以通过CSS样式进行自定义，以改变水平线的颜色、宽度、样式等。

```html
<style>
    hr {
        color: red; /* 修改水平线颜色 */
        border-style: dotted; /* 修改水平线样式 */
        border-width: 2px; /* 修改水平线宽度 */
    }
</style>
```

总之，`<hr>`标签是一种简单而常用的方式，用于在HTML中创建水平分隔线，以增强页面的可读性和视觉层次感。

## HTML段落

在HTML中，段落（Paragraph）是一种用于组织和呈现文本内容的基本结构。段落可以通过使用`<p>`标签来定义，该标签表示一个独立的文本段落。

以下是一个简单的示例，演示如何在HTML中创建段落：

```html
<!DOCTYPE html>
<html>
<head>
    <title>HTML段落示例</title>
</head>
<body>
    <p>这是第一个段落。</p>
    <p>这是第二个段落。</p>
    <p>这是第三个段落。</p>
</body>
</html>
```

在上面的示例中，我们使用了三个`<p>`标签来创建三个不同的段落。每个`<p>`标签定义了一个独立的文本块，浏览器会在段落之间添加默认的空白行来分隔它们。

段落通常用于将相关的文本组织成逻辑单元，使页面内容更具结构和可读性。您可以在段落中放置任意文本内容，包括文字、链接、图像等。通过CSS样式，您还可以自定义段落的外观，如文本颜色、字体大小、行距等。

```html
<style>
    p {
        font-size: 16px;
        color: #333;
        line-height: 1.5;
    }
</style>
```

总之，`<p>`标签是在HTML中创建段落的标准方式，它帮助将文本内容分组，提高页面的可读性和排版效果。

## HTML 折行

在HTML中，可以使用`<br>`标签来创建折行（Line Break）。`<br>`标签是一个自闭合标签，用于在文本中插入换行符，将文本拆分成多行。

以下是一个示例，演示如何在HTML中使用`<br>`标签创建折行：

```html
<!DOCTYPE html>
<html>
<head>
    <title>HTML折行示例</title>
</head>
<body>
    <p>这是一行文本。<br>这是折行后的文本。</p>
</body>
</html>
```

在上面的示例中，我们在段落中使用了`<br>`标签，将文本分成两行。`<br>`标签不需要闭合，它会在文本中插入一个换行符，使得文本在页面上换行显示。

需要注意的是，过度使用`<br>`标签可能会导致代码难以维护和排版混乱。通常情况下，更好的做法是使用CSS样式来控制文本的排版和换行，例如设置`white-space: pre-line;`来保留空格和换行。

```html
<style>
    p {
        white-space: pre-line;
    }
</style>
```

这将在保留文本的空格和换行的同时，根据需要进行自动换行。这种方式更利于维护和页面排版的灵活性。

## HTML 文本格式化

HTML提供了多种标签和属性来进行文本格式化，以改变文本的样式、排列和呈现。以下是一些常见的HTML文本格式化方法：

1. **粗体和斜体：**
   - `<b>`标签：表示粗体文本。
   - `<strong>`标签：表示重要的文本，通常呈现为粗体，但语义上更强调。
   - `<i>`标签：表示斜体文本，但没有特定的语义。
   - `<em>`标签：表示强调的文本，通常呈现为斜体，强调重要性或语气。

```html
<p>这是一段<b>粗体</b>文本，<strong>重要</strong>的内容。</p>
<p>这是一段<i>斜体</i>文本，<em>强调</em>的部分。</p>
```

2. **下划线和删除线：**
   - `<u>`标签：表示带有下划线的文本。
   - `<s>`标签：表示带有删除线的文本。

```html
<p>这是一段<u>带下划线</u>的文本，<s>带删除线</s>的内容。</p>
```

3. **上标和下标：**
   - `<sup>`标签：表示上标文本，常用于表示数学公式中的上标。
   - `<sub>`标签：表示下标文本，常用于表示数学公式中的下标。

```html
<p>2<sup>3</sup> 表示 2 的 3 次方，H<sub>2</sub>O 表示水分子。</p>
```

4. **换行和段落：**
   - `<br>`标签：表示换行，用于在文本中插入一个换行符。
   - `<p>`标签：表示段落，用于将相关内容组织成段落。

```html
<p>这是第一段。<br>这是第二段。</p>
<p>这是另一个段落。</p>
```

5. **文本对齐：**
   - `text-align`属性：用于控制文本的对齐方式（left、right、center、justify）。
   - `<div>`标签：用于创建一个块级容器，可以使用`text-align`属性来设置内部文本的对齐方式。

```html
<p style="text-align: center;">居中对齐的文本。</p>
<div style="text-align: right;">
    这是右对齐的文本。
    <p>这是内部的段落。</p>
</div>
```

6. **字体大小和颜色：**
   - `<font>`标签（不推荐使用）：用于设置字体、颜色、大小等样式。更推荐使用CSS样式来控制文本样式。

```html
<p style="font-size: 18px; color: blue;">这是18px大小的蓝色文本。</p>
```

这些只是一些常见的HTML文本格式化方法。为了更好地控制文本样式，建议使用CSS来进行格式化，以便在整个网站上实现一致的视觉效果。

## HTML 链接

在HTML中，链接（Link）是用于在不同网页之间或同一网页内创建导航和跳转的元素。链接通常是用超链接标签`<a>`（Anchor）来表示的。下面是如何在HTML中创建链接的示例：

1. **外部链接：**
   外部链接用于链接到其他网站的页面。在`<a>`标签的`href`属性中指定目标网址。

```html
<a href="https://www.example.com">访问示例网站</a>
```

2. **内部链接：**
   内部链接用于在同一网站内部不同页面之间进行跳转。您可以使用相对路径来指定目标页面的位置。

```html
<a href="/about.html">关于我们</a>
```

3. **锚点链接：**
   锚点链接用于在同一页面内部不同部分之间进行跳转。在`href`属性中使用`#`后跟目标元素的`id`。

```html
<a href="#section1">跳转到第一部分</a>
```
```html
<!-- 目标位置 -->
<h2 id="section1">第一部分标题</h2>
```

4. **电子邮件链接：**
   您可以使用`mailto:`前缀创建一个电子邮件链接，使用户能够点击链接发送邮件。

```html
<a href="mailto:contact@example.com">联系我们</a>
```

5. **下载链接：**
   您可以使用`download`属性指定文件的下载链接，浏览器将提示用户下载文件。

```html
<a href="/files/document.pdf" download>下载PDF文档</a>
```

6. **新标签页打开链接：**
   使用`target="_blank"`属性可以让链接在新的浏览器标签页中打开。

```html
<a href="https://www.example.com" target="_blank">在新标签页打开示例网站</a>
```

以上示例展示了一些不同类型的链接。`<a>`标签还可以与其他HTML元素结合使用，以创建更丰富的链接效果。链接对于构建导航、引导用户和提供更多信息都非常有用。
